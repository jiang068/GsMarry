# 娶群友插件

一个有趣的群聊互动插件，让群友们可以每天随机"娶群友"！

## 功能特性

-  **随机匹配**：从群友中随机选择"老婆"
- ⏰ **每日限制**：每人每天只能娶一次
- � **强娶功能**：可指定用户强娶，带成功率
- 💔 **离婚功能**：可解除当日婚姻关系
- �🖼️ **头像显示**：可显示被娶用户的头像
- ⚙️ **灵活配置**：支持自定义消息和参数

## 使用命令

- `娶群友` / `waifu` / `wife` - 随机娶群友
- `强娶@用户` - 强制娶指定用户
- `离婚` - 解除婚姻关系
- `今日cp` - 查看今日CP榜
- `强娶记录` - 查看强娶记录
- `娶群友设置` - 查看配置信息

## 安装

1. 将 `gs_marry` 文件夹放入 GsCore 插件目录
2. 重启 GsCore 即可使用
